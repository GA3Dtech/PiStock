# Contributing to PiStock

## English version

Thank you for your interest in PiStock! Here is how to contribute.

### Before contributing

Any contribution implies acceptance of the **Contributor License Agreement (CLA)**.  
→ Please read [CLA.md](./CLA.md) before opening a Pull Request.

### Contribution process

1. Fork the repository
2. Create a branch from `main`: `git checkout -b feat/my-feature`
3. Make your changes and commit them
4. Open a Pull Request targeting `main`
5. If it's your first contribution, the CLA bot will ask you to sign the CLA by commenting on the PR:

   > `I have read the CLA Document and I hereby sign the CLA`

6. Once the CLA is signed and the PR approved, it will be merged.

### General guidelines

- Follow the existing code style
- Add tests where applicable
- Clearly describe your changes in the PR description

### Questions?

Open an [issue](https://github.com/pistock-org/PiStock/issues) for any question.
